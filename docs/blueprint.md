# **App Name**: eMenu Digital Hub

## Core Features:

- Dashboard Overview: Display key metrics like total categories, active products, published pages, and today's orders.
- Menu Management: Enable restaurant owners to add, edit, and categorize products within their digital menu.
- Promotions & Coupons: Allow users to create and manage promotions and coupons for their menu items.
- Order Analytics: Provide a visual representation of order analytics for a specified time period.
- Inventory Alerts: Send notifications for low stock items, prompting reordering or review.
- Performance Reporting: Display a summary of weekly revenue and average order value.

## Style Guidelines:

- Primary color: Light teal (#A7E7E1) to evoke a fresh and modern feel.
- Background color: Off-white (#F5F7FB) for a clean and airy aesthetic.
- Accent color: Soft pink (#F4B9DE) for highlighting key elements and creating a playful contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif that provides both a modern look and a touch of warmth.
- Use flat, minimalist icons in the primary color to represent different menu categories and actions.
- Design a card-based layout for displaying data and information, with rounded corners and subtle shadows for depth.
- Implement subtle animations for transitions and interactions to provide a smooth user experience.
